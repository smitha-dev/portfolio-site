Two-Pass Interpreter

This project is a two-pass interpreter for a simple assembly language, designed for the Computer Organization course (CSCI 355) at the University of Findlay. It reads an assembly program from a file (tiny.txt), translates it into machine language, stores it in simulated memory, and interprets and executes the instructions. The instruction set supports input/output, basic arithmetic, and data movement within a simulated memory structure.
